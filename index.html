About CassidySpectra

CassidySpectra is a powerful, all-purpose chatbot framework built with TypeScript, supporting platforms like Facebook Messenger, Discord, and more. As a fork of CassidyRedux, it offers enhanced features, superior performance, and the best tooling for developers. Created and maintained by Liane Cagara.

Explore on GitHub

Key Features

Advanced TypeScript Tooling

Enjoy top-tier TypeScript support with autocomplete and type safety for seamless development.

Multi-Platform Support

Run your bot on Facebook Messenger, Discord, and other platforms with ease.

Rich Command System

Create and manage commands with premade templates and automatic styling.

Extensible Plugins

Integrate APIs, add plugins, and customize with MongoDB and idle game support.

Get Started

Ready to build your own chatbot? Follow the Cassidy Bot Tutorial to set up CassidySpectra on Railway, configure Facebook Messenger, and deploy your bot. Check out the command creation guide to start coding.
